What is built

Three working features, two real roles

Log in as a parent or teacher right now. Every feature below is live with a real Flask API backend, not placeholder data.

💬
Hero feature

Real-time Parent-Teacher Messaging

Threaded message bubbles between parent and teacher. Messages are stored in a real database and poll every 4 seconds, so a message sent on the teacher side appears on the parent side without a page refresh. Unread badge updates live in the nav.

📊
Supporting

Live Student Performance Dashboard

Six subject grade cards (Math, English, Science, History, Art, PE) with letter grades, numeric scores, color-coded status bars, and a Chart.js radar chart. The teacher can update any grade and the parent sees it within seconds via polling.

📢
Supporting

Activity and Announcements Feed

A chronological feed of academic milestones, school events, attendance notes, and teacher announcements. Teachers post from a compose panel and the item appears instantly at the top of the parent's feed.
