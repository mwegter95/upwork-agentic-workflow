Production PostgreSQL path

The demo runs on SQLite with a fully PostgreSQL-compatible schema (standard SQL types, no SQLite-isms, full foreign key and index definitions). Moving to a hosted Postgres instance is a driver swap from node:sqlite to pg + pg-pool. The schema and all queries transfer unchanged.
