Money Correctness
Obsessive about money. Integer cents throughout.
No floats on any money path. Every amount is stored, computed, and transmitted as integer minor units (cents/centavos). Zod enforces this at every API boundary. Discriminated union types carry the currency context.
Face Value $8,750.00 875000 cents
Advance (85%) $7,437.50 743750 cents
Fee (3%) $262.50 26250 cents
Reserve holdback $1,050.00 105000 cents
INV-002 Northern Freight Co. Computed in integer arithmetic only.
Type definitions
// Discriminated union for currency
type USD = { currency: 'USD'; amount: number };
type BRL = { currency: 'BRL'; amount: number };
type Money = USD | BRL;

// Zod schema (all amounts integer)
const InvoiceSchema = z.object({
  faceValueCents: z.number().int().positive(),
  advanceCents: z.number().int().nonnegative(),
  feeCents: z.number().int().nonnegative(),
  reserveCents: z.number().int().nonnegative(),
});
