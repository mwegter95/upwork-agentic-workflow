Your Questions, Answered

From the brief, directly

Can you launch within one week?
Yes. Seven days from go-ahead to live site. The 7-day plan above is the commitment, not an estimate.
Can you stay within the Phase 1 budget?
Yes. $1,100 fixed, no overages. Rocket.net hosting for the first two months is included in that price.
Will the site be easy for the owner to edit?
Yes. WordPress with Bricks Builder gives you a visual drag-and-drop editor for text, photos, pricing, FAQs, and service areas. No developer needed for routine updates. The handoff on Day 7 includes a walkthrough.
How do you optimize Core Web Vitals?
ShortPixel compresses every image on upload. Rocket's built-in CDN and page caching eliminate server latency. Lazy loading defers off-screen images. Plugin count is kept minimal (your exact list, nothing extra). Lighthouse 90+ is the target before we go live.
Will you configure Analytics, Search Console, and schema?
Yes. GA4 and GSC via Google Site Kit on Day 5. All four schema types (LocalBusiness, Service, FAQ, Review) embedded via Rank Math before launch. CTA clicks and form submissions tracked as GA4 events.
Can Phase 2 be added without rebuilding?
Yes. The Phase 1 site is architected for Phase 2 from the start. City landing pages, the learning center, and the estimate calculator are additive layers on the same WordPress install, same theme, same SEO foundation.
